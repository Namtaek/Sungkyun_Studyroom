# Sungkyun_Studyroom

## 2021-01 성균공부방 강의 자료

2021년 겨울에 성균공부방에서 '통계적 데이터 분석 입문'이라는 이름으로 강의한 내용에 대한 자료입니다. 통계/회귀분석의 이론에 대한 교안과 R 실습을 담고 있습니다. 수강한 학생들이 대부분 비전공자이기 때문에, 이론적으로 깊은 내용을 다루지는 않았습니다. 하지만 각각 스텝에서 중요한 아이디어와, 어떤 실제 현실에 적용이 가능한지를 전달하기 위해 노력했습니다.
